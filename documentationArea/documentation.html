Luau Review Documentation

Learn how Luau Review analyzes Roblox scripts and helps improve your code.

Getting Started

1. Paste your Luau code into the editor.
2. Click Analyze Code.
3. Review detected issues.
4. Apply the recommended improvements.

How Analysis Works

Syntax Analysis

Checks if your script contains errors that could stop it from running.

Performance Analysis

Finds inefficient code, expensive loops, and possible lag sources.

Security Analysis

Checks RemoteEvents, RemoteFunctions, and unsafe client communication.

Supported Checks

Performance

  • Infinite loops
  • Heavy RenderStepped usage
  • Repeated object searching
  • Large scripts

Memory

  • Unused connections
  • Possible memory leaks
  • Uncleaned objects

Security

  • RemoteEvent validation
  • Unsafe client trust
  • Missing argument checks

Luau Best Practices

Use task instead of old functions

Bad:
wait(1)

Good:
task.wait(1)

Enable Strict Mode

--!strict

Strict mode improves type checking and catches errors earlier.
